Automation with Agents

It's great that Wingman can stand up an app, but the goal is to let it join the business. Point it at real tasks and invite it to add value where teams feel pressure.

For this next task we're going to use an AI agent, not to code and app but to operate it and add value, like a useful little assistant!

Clone Your Agent

Set up a new agent app in Wingman and point it at the starter repo.

  1. Go to New App in Wingman
  2. Clone the repo:
    https://gitea.pages.otherstuff.ai/honest-ivory-thicket/speedrun-lite-agent-starter.git
  3. Give your agent a friendly name

Give It Permission

Your agent needs delegation rights so it can read and write on your behalf.

  1. Go to Settings in the Wingman app
  2. Copy the Bot npub
  3. Go to your app and click the avatar
  4. Select Delegates, paste the code and give it permission to read and write

Put Wingman to work for you!

Let the agent read your todos, analyse what's coming up, and write you a summary — daily and weekly — without you lifting a finger.

Agent feedback
I want you to do three things in succession ensuring that each is done before you move on. Do not use sub agents. 1. Review the agent conect spec in the conenct json which will tell you how to read my todos and update summaries. 2. Please read my current todos for ownerPubkey. 3. Please review all my upcoming todos, and write back daily summary for me to review in the app.
